# Product Context

## Target Market

SME owners (1-50 employees) who need a professional web presence but cannot afford traditional agencies ($10K-$50K+) and are frustrated by the limitations of DIY builders (Wix, Squarespace). They want agency-quality results at SaaS prices with zero technical effort.

## Pricing Model

| Tier | Monthly | Includes |
|------|---------|----------|
| **Starter** | $99 | 5-page site, 3 core blocks, basic SEO, monthly content update |
| **Growth** | $249 | 15-page site, 10+ blocks, blog, analytics, weekly content updates |
| **Scale** | $499 | Unlimited pages, custom blocks, integrations, daily Digital Team ops |

All tiers include hosting, SSL, CDN, and the ejectable repo guarantee.

## Competitive Landscape

| Competitor | Weakness FullStop Exploits |
|-----------|---------------------------|
| **Wix/Squarespace** | Vendor lock-in. No code ownership. Export = broken HTML dump. |
| **WordPress** | Plugin hell. Security burden on client. Needs ongoing dev maintenance. |
| **Traditional Agencies** | $10K-$50K upfront. 8-16 week timelines. Ongoing retainer for changes. |
| **AI Website Builders** | Generates throwaway HTML. No CMS. No content management system. No data model. |

## Moat

1. **Data Sovereignty**: Client owns a real git repo, not a platform project. Ejectable at any time.
2. **Block Architecture**: Reusable, composable blocks compound across clients. Factory gets faster with scale.
3. **Two-Tier AI**: Separate build agents (Factory) and operate agents (Digital Team) prevents scope creep and enables specialization.
4. **Controlled Factory**: 6-layer Repo OS makes AI execution deterministic, auditable, and repeatable. Not prompt-and-pray.

## Key User Journeys

1. **Onboarding**: Client fills Business Model Canvas form -> Factory agents build site in <24hrs -> Client reviews in staging -> Deploy
2. **Content Update**: Client requests change via dashboard -> Digital Team agent executes -> Verification layer validates -> Auto-deploy
3. **Ejection**: Client cancels -> Downloads git repo + Docker Compose file -> Runs independently. Zero migration cost.
